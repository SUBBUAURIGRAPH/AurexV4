## How to Use

### 1. Access the Plugin

The plugin is installed globally and available via npm:

```bash
# Verify installation
npm list -g @aurigraph/claude-agents-plugin

# Expected output:
# C:\Users\SubbuJois\AppData\Roaming\npm
# └── @aurigraph/claude-agents-plugin@2.2.0
```

### 2. Run Skills

Use the installed plugin with any of the developer tools skills:

```bash
# Security scanning
@aurigraph/claude-agents-plugin scan-security

# Performance profiling
@aurigraph/claude-agents-plugin profile-code

# Documentation generation
@aurigraph/claude-agents-plugin generate-docs

# Comprehensive code review
@aurigraph/claude-agents-plugin comprehensive-review

# Code quality analysis
@aurigraph/claude-agents-plugin analyze-code

# Test execution
@aurigraph/claude-agents-plugin run-tests
```

### 3. Configuration

Create a `.env` file in your project:

```env
AURIGRAPH_API_KEY=your_api_key_here
DEBUG=false
LOG_LEVEL=info
```
