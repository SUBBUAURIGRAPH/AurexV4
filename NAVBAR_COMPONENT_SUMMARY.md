## Design Specifications

### Colors
- Primary: #667eea (Purple)
- Secondary: #764ba2 (Dark Purple)
- Background: rgba(15, 15, 30, 0.95)
- Text: #fff (White)
- Accent: #999 (Light Gray)

### Typography
- Logo: 24px, 700 weight
- Menu items: 14px, 500 weight
- Dropdown items: 13px, 400 weight

### Dimensions
- Navbar height: 60px (fixed)
- Gap between menu items: 30px
- Dropdown min-width: 220px
- Mobile breakpoint: 768px

### Animations
- Dropdown open/close: 300ms
- Chevron rotation: 300ms
- Hover effects: 200-300ms
- All use ease timing function

## Integration Quick Start

### 3-Step Process

1. **Copy Code**
   - Open `nav-template.html`
   - Copy all the code

2. **Paste into Page**
   - Add at top of `<body>` tag
   - All styles and scripts included

3. **Adjust Layout**
   - Add `margin-top: 60px` to main content
   - No other changes needed
