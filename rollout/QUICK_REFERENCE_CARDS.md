# Quick Reference Cards - Aurigraph Agents

**Format**: Print-ready, single-page reference cards for each agent
**Size**: 8.5" x 11" (Letter) or A4
**Distribution**: Print and distribute to team members, post near workstations

## How to Use These Cards

1. **Print**: Each card section below is designed for printing
2. **Cut**: Each agent gets one full page
3. **Distribute**: Give to relevant team members
4. **Post**: Place near workstations or in common areas
5. **Digital**: Share PDF versions via email or Slack

   - Print quality: High

   - Cardstock (110 lb / 200 gsm) for durability

   - Plain paper for quick reference
