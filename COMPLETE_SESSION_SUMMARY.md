## How to Use

### Jeeves4Coder Agent (Aurigraph Ecosystem)

Access via agent orchestration:
```javascript
const orchestrator = new AgentOrchestrator();
const review = await orchestrator.invokeAgent({
  agentId: 'jeeves4coder',
  skill: 'code-review',
  input: { code: sourceCode }
});
```

Or via plugin:
```bash
node plugin/environment-loader.js init
# Jeeves4Coder available through initialized environment
```

### Jeeves4coder Agent (Claude Code)

Use directly in Claude Code:
```
Jeeves4coder, review this code for security issues:
[your code]
```

Or automatic trigger:
```
I need code review before merging:
[your code]
# Claude Code automatically invokes Jeeves4coder
```
