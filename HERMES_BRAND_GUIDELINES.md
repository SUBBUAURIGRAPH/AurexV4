## TYPOGRAPHY

### Font Family
**Primary**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial

### Type Scales

**Headings**
- H1: 3.5rem (56px) - Landing page hero
- H2: 2.5rem (40px) - Section titles
- H3: 1.5rem (24px) - Card titles
- H4: 1.25rem (20px) - Subsections

**Body Text**
- Large: 1.125rem (18px) - Important content
- Regular: 1rem (16px) - Main body text
- Small: 0.875rem (14px) - Secondary content
- Extra Small: 0.75rem (12px) - Labels, metadata

**Font Weights**
- Light: 300 (not common)
- Regular: 400 (body text)
- Medium: 500 (emphasis)
- Semibold: 600 (labels, strong content)
- Bold: 700 (headings, important)

## LOGO USAGE

### Logo Formats

**Horizontal Logo** (Preferred)
```
📈 HERMES
```
Usage: Header navigation, main branding

**Icon Only**
```
📈
```
Usage: Favicon, app icons, badges

**Vertical Logo**
```
📈
HERMES
```
Usage: Footer, sidebars

### Clear Space
Minimum clear space around logo: 0.5x the height of the "H" in HERMES

### Logo Do's ✅
- Use on solid backgrounds
- Maintain proportions
- Use complete logo or icon only
- Ensure sufficient contrast
- Scale proportionally

### Logo Don'ts ❌
- Don't stretch or skew
- Don't rotate or tilt
- Don't place on busy backgrounds
- Don't add effects or shadows
- Don't separate icon from text (except favicon)

## IMAGERY & PHOTOGRAPHY

### Style Guide

**Charts & Graphs**
- Clean, minimal design
- Use brand purple for positive metrics
- Use success green for gains
- Use error red for losses
- Include clear legends and labels

**Photography**
- Professional, modern aesthetic
- Show real traders and advisors
- Display diverse demographics
- Emphasize technology and innovation
- Use natural lighting when possible

**Icons**
- Emojis for primary UI elements
- Simple, 24x24px for secondary icons
- Consistent line weight (2px)
- Clear, recognizable symbols

### Image Specifications

**Hero Images**: 1920x1080px minimum, 16:9 aspect ratio
**Card Images**: 800x400px, 2:1 aspect ratio
**Icons**: 24x24px-256x256px, scalable
**Social Media**: Platform-specific dimensions

## VOICE & TONE

### Brand Voice

**Professional Yet Approachable**
- We use clear, jargon-free language when possible
- We explain complex concepts simply
- We're friendly but not overly casual

**Confident & Knowledgeable**
- We stand behind our product and promises
- We share expertise without being preachy
- We acknowledge limitations honestly

**Empowering & Supportive**
- We focus on user success and growth
- We celebrate wins and milestones
- We provide guidance without being prescriptive

### Tone Examples

**In Marketing**:
"Trade smarter with AI-powered insights. See better returns. Save hours each week."

**In Support**:
"Great question! Here's how to set up your first alert in 3 steps..."

**In Error Messages**:
"Something went wrong. Don't worry—our team is looking into it. Refresh the page or contact us if it persists."

## MESSAGING FRAMEWORKS

### Problem-Solution-Benefit (PSB)

**Problem**: Manual trading analysis is time-consuming and emotional
**Solution**: Hermes provides AI-powered insights and automated execution
**Benefit**: Better trades, higher returns, lower stress, saved time

### Feature → Benefit → Value

Example:
- **Feature**: 50ms execution latency
- **Benefit**: You get the best prices on your trades
- **Value**: Save $1,000s annually in missed executions

### Three-Pillar Messaging

1. **Intelligence** - AI-powered insights you can trust
2. **Speed** - Execution that doesn't miss opportunities
3. **Security** - Enterprise protection for your assets
