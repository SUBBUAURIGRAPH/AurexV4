## GNN Agent Capabilities

### 1. Graph Neural Network Architectures

**Supported GNN Models**:
- ✅ **GCN** (Graph Convolutional Networks)
  - Standard graph convolution
  - Message passing on neighborhoods

- ✅ **GAT** (Graph Attention Networks)
  - Attention-based neighbor aggregation
  - Learned importance weights

- ✅ **GraphSAGE** (Graph Sample and Aggregate)
  - Inductive learning
  - Mini-batch training

- ✅ **GIN** (Graph Isomorphism Networks)
  - Expressive graph learning
  - Function approximation

- ✅ **MPNN** (Message Passing Neural Networks)
  - General message passing framework
  - Flexible node/edge updates

- Optimal decision patterns
- Problem-solving strategies

## 8 Core GNN Skills

### Skill 1: `graph-constructor`
**Purpose**: Build graph representations from data

**Inputs**:
- Data sources (CSV, JSON, databases, APIs)
- Graph schema
- Feature definitions

**Process**:
- Load and preprocess data
- Extract nodes and edges
- Compute node features
- Compute edge features
- Build and validate graph

**Output**: Graph object with features

**Use Cases**:
- Social network analysis
- Knowledge graph construction
- Supply chain networks
- Trading networks

---

### Skill 2: `gnn-trainer`
**Purpose**: Train GNN models on graph data

**Inputs**:
- Graph data
- Labels/targets
- Model architecture
- Training parameters

**Process**:
- Initialize GNN model
- Define loss function
- Train on mini-batches
- Validate on held-out data
- Save best model

**Output**: Trained model with metrics

**Training Features**:
- Early stopping
- Learning rate scheduling
- Gradient clipping
- Batch normalization

---

### Skill 3: `heuristic-learner`
**Purpose**: Learn optimization heuristics via RL

**Inputs**:
- Problem instances
- GNN architecture
- Reward function
- RL parameters

**Process**:
- Initialize policy network
- Collect trajectories
- Compute rewards
- Update policy
- Evaluate on test set

**Output**: Learned heuristic policy

**Applications**:
- TSP solving
- VRP solving
- Task scheduling
- Resource allocation

---

### Skill 4: `graph-attention-optimizer`
**Purpose**: Use attention mechanisms for optimization

**Inputs**:
- Problem graph
- Objective function
- Constraints
- Attention parameters

**Process**:
- Encode graph with attention
- Learn node importances
- Construct solution sequentially
- Optimize attention weights
- Validate constraints

**Output**: Optimized solution

**Features**:
- Multi-head attention
- Learned importance weights
- Constraint satisfaction
- Interpretability

---

### Skill 5: `combinatorial-solver`
**Purpose**: Solve combinatorial optimization problems

**Inputs**:
- Problem instance
- Constraints
- Objective metrics
- Solver parameters

**Process**:
- Formulate as graph problem
- Run GNN solver
- Apply post-processing
- Verify solution
- Return metrics

**Output**: Optimized solution with metrics

**Problem Types**:
- TSP (Traveling Salesman)
- VRP (Vehicle Routing)
- Job Scheduling
- Resource Allocation

---

### Skill 6: `graph-representation-learner`
**Purpose**: Learn meaningful graph embeddings

**Inputs**:
- Graph data
- Embedding dimension
- Learning task
- Parameters

**Process**:
- Define loss function
- Train embedding model
- Validate embeddings
- Extract representations
- Analyze embedding space

**Output**: Node embeddings & analysis

**Methods**:
- Contrastive learning
- Supervised learning
- Self-supervised learning
- Autoencoders

---

### Skill 7: `graph-anomaly-detector`
**Purpose**: Detect anomalies in graph data

**Inputs**:
- Graph data
- Normal behavior baseline
- Anomaly threshold
- Detection parameters

**Process**:
- Learn normal pattern
- Compute reconstruction error
- Identify deviations
- Rank anomalies
- Generate alerts

**Output**: Anomaly scores & reports

**Applications**:
- Fraud detection
- Network intrusion detection
- Supply chain anomalies
- Trading anomalies

---

### Skill 8: `temporal-graph-forecaster`
**Purpose**: Forecast graph evolution over time

**Inputs**:
- Temporal graph sequence
- Forecast horizon
- Model parameters
- External features

**Process**:
- Encode temporal dynamics
- Learn evolution patterns
- Forecast future states
- Quantify uncertainty
- Validate forecasts

**Output**: Predictions with confidence

**Applications**:
- Market trend forecasting
- Network growth prediction
- Traffic prediction
- Supply chain forecasting
