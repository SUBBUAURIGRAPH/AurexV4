# HMS J4C Agent Integration Guide

## Quick Start

### Installation
npm install --legacy-peer-deps
cd backend && npm install --legacy-peer-deps && cd ..

### Running HMS Backend
npm run dev:backend

### Checking J4C Health
npm run j4c:health

### Running J4C Agent
Option 1: npm run j4c:discover
Option 2: npx ts-node j4c-hermes-adapter.ts
Option 3: docker-compose -f docker-compose.hermes.yml up -d
