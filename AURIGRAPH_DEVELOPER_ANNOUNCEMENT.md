## 🚀 HOW TO GET STARTED (5 Minutes)

### Step 1: Update Your Project

Choose one of these methods:

**Option A: Git Submodule (Recommended)**
```bash
cd /your/aurigraph/project
git submodule add git@github.com:Aurigraph-DLT-Corp/glowing-adventure.git .claude
git submodule update --init --recursive
git add .gitmodules .claude
git commit -m "Add Aurigraph agents architecture"
```

**Option B: Direct Clone**
```bash
cd /your/aurigraph/project
git clone git@github.com:Aurigraph-DLT-Corp/glowing-adventure.git .claude
```

### Step 2: Verify Installation

```bash
# Check if agents are installed
ls -la .claude/agents/

# You should see:
# - dlt-developer.md
# - trading-operations.md
# - devops-engineer.md
# - jeeves4coder.md
# ... and 9+ more
```

### Step 3: Start Using an Agent

```bash
# In Claude Code, try:
@jeeves4coder "Review this function"

# Or try another agent:
@devops-engineer "Deploy to dev4"
@project-manager "Create a JIRA ticket"
```
