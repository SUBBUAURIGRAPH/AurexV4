**Most Common Questions (answered):**

**Q: Do I need to do anything?**
A: Just run the git submodule command above. Takes 1 minute.

**Q: Will this slow down my project?**
A: No! Agents run in Claude Code, zero impact on your codebase.

**Q: Can I use this in production deployments?**
A: Agents are assistants. Always review their output. Don't blindly deploy.

**Q: What if I have an issue?**
A: Respond in this thread or email agents@aurigraph.io. We're here to help!

**Q: Can I customize agents for my team?**
A: Yes! Copy agent files and modify. We can guide you.

**Q: When do I get Strategy Builder?**

**Q: What languages do agents support?**
A: Jeeves4Coder supports 10+ languages (JavaScript, Python, Java, Go, Rust, SQL, Ruby, PHP, Kotlin, C/C++)

**Q: Is this mandatory?**
A: No, but we think you'll love it! Optional, but highly recommended.
